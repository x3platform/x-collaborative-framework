X3Platform.Services.Host (Windows 服务) 宿主程序 帮助手册

============================================
Windows 命令
============================================

1.安装服务
X3Platform.Services.Host.exe install

2.卸载服务
X3Platform.Services.Host.exe uninstall

3.启动服务
net start "X3Platform Services Host"

停止服务
net stop "X3Platform Services Host"

注:"X3Platform Services Host" 为 X3Platform.Services.Host.exe.config 配置文件中 Services.ServiceName 的值.
